# Simple Linear Regression

## Objective

The objective of this project is to predict the height of a person using their weight through Simple Linear Regression.

---

## Dataset

**Dataset Name:** Height-Weight Dataset

Features:

- Weight (Independent Variable)

Target:

- Height (Dependent Variable)

---

## Workflow

1. Import the dataset.
2. Explore the data.
3. Split the dataset into training and testing sets.
4. Train the Linear Regression model.
5. Predict the output.
6. Visualize the regression line.
7. Evaluate the model.

---
